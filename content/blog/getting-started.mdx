# Getting Started with Next.js

Welcome to my first blog post! In this article, I'll share my experience getting started with Next.js.

## Why Next.js?

Next.js is a powerful React framework that makes it easy to build full-stack web applications.

## Key Features

- **Server-Side Rendering** - Better SEO and initial load performance
- **File-based Routing** - Intuitive and easy to understand
- **API Routes** - Build your backend in the same project
- **Image Optimization** - Automatic image optimization

## Getting Started

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```
